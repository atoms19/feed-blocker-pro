## instructions to install (detailed)

if you have git installed on your computer you can clone this repo 
```bash
git clone https://github.com/atoms19/feed-blocker-pro.git
```
then you can skip to step 3 , and select the folder which you cloned the repo to in step 4

for non git users follow the instructions below step by step

**step 1** : download the repo
first you must download this repo as a zip file and extract it to a folder 
u can download the repo by clicking the green code button on the top right of the page(assuming you are viewing this on github)
then click the `download zip` option and save the file to your computer

**step 2**: extract the zip file
after you have downloaded the zip file , right click on the file and select `extract all` to extract the contents of the zip file to a folder
now open google chorme (or any chromium based browser (brave, edge etc)) and go to the extensions page by typing `chrome://extensions` in the address bar and pressing enter

**step 3**: turn on developer mode 
one you are there you will see a toggle switch in the top right corner of the page 
to turn on developer mode click on the toggle switch and it will turn blue

after you have turned on developer mode you will see three buttons appear on the page
`load unpacked` , `pack extension` and `update`

**step 4**: load the extension
click on the `load unpacked` button and a file explorer window will open
select the folder which you extracted the zip file to and click on the `select folder` button
after you have selected the folder the extension will be loaded and you will see it in the list of the extensions page 
now you can click on the toggle switch next to the extension to turn it on and start blocking social media feeds

## removing the extension
to remove the extension you can simply click on the `remove` button next to the extension in the extensions page and confirm the removal by clicking on the `remove` button in the pop-up window
you can delete the folder which you extracted the zip file to as well if you want to free up some space on your computer


## updating the extension
this is the beta stage of the extension so there will be many updates
and in the future it will be a lot easier once i get it on webstore 

for now updates are for those who have cloned the repo using git
if you have cloned the repo using git you can simply pull the latest changes from the repo by
```bash
git pull origin main
```
after you have pulled the latest changes you can go to the extensions page and click on the `update` button to update the extension to the latest version
or just reload the extension by turning it off and on again

>please note that if you have downloaded the repo as a zip file you will have to repeat the installation process to get the latest updates as there is no way to update the extension without git
